# Contributing

This repository contains documentation and the OpenAPI spec for the Nakordoni Developer API.

## Found an error in the docs?

- Open a GitHub issue with the product name and what's wrong
- Or open a support ticket in your [developer dashboard](https://nakordoni.eu/en/developers/tickets)

## OpenAPI spec (`openapi.yaml`)

The spec is generated from the live endpoint map. If you notice a missing parameter or wrong description, open an issue — we update the live docs and regenerate.

## Code samples

PRs adding client examples (Go, Ruby, Kotlin, Swift, etc.) are welcome. Keep them minimal and focused on the API call, not on parsing the response.
